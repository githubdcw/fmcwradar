# FMCW Radar
This project provides information and resources for the design of Frequency-Modulated Continuous Wave (FMCW) radar systems.

- 2.4GHz band, 2400-2500MHz = 100MHz bandwidth

  - 5725 - 5850 เมกะเฮิรตซ 1 W e.i.r.p. ได้รับยกเว้นใบอนุญาต ทั่วไป - รายการ 5 ภาคผนวก ง
